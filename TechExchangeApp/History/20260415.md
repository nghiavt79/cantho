## Quyết định kỹ thuật

| Vấn đề | Giải pháp |
|---|---|
| Reuse CMS logic mà không duplicate | `internal static` method trên `SanPhamCNTBController` |
| Create + Edit dùng cùng view | `isEdit = Model.ID > 0` — detect trong view, thay đổi action URL và button text |
| `@section` trong partial không render | Inline scripts trực tiếp vào `@section Scripts` của view chính |
| Form tag conflict `asp-action` + `action` | Chỉ dùng `method="post" action="@formAction"`, bỏ `asp-action`/`asp-controller` |
| CKEditor UMD 43 lỗi (thiếu CSS, sai plugin) | Chuyển sang super-build 41.4.2 giống CMS |
| Build lỗi StaticWebAssets duplicate | MSBuild Target normalize path trong `.csproj` |
| Edit không reset status không cần thiết | Status=3 → 2; các status khác giữ nguyên |
